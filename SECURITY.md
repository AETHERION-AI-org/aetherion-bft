# Security Policy

## Reporting a vulnerability

If you discover a security vulnerability in AETHERION BFT, please report it responsibly.

- **Do not** open a public issue for security-sensitive reports.
- Email the maintainers with a detailed description, reproduction steps and, where possible, a
  suggested remediation.
- Please allow a reasonable window for a fix to be prepared and deployed across the validator set
  before any public disclosure.

Responsible disclosures that help keep the Aetherion Network and its participants safe are greatly
appreciated.
